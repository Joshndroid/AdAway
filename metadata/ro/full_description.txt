Un ad blocker care foloseste fisierul 'hosts'
Fisierul 'hosts' este un fisier de sistem ce contine legaturi intre numele de domenii si adresele ip
Cand o aplicatie solicita o reclama dintr-un domeniu aflat in fisierul 'hosts' va fi redirectionata la adresa locala 127.0.0.1 care nu face nimic.
Puteti descarca fisiere predefinite cu nume de domenii direct din aplicatie pentru a le incorpora direct in fisierul de system 'hosts. De asemenea este posibil sa folositi fisiere si surse proprii pentru a permite sau bloca anumite  domenii.
Exista optiuni pentru a rula un server web local ce va raspunde domeniilor blocate cu redirectionari la adresa ip aleasa de dumneavoastra.

Necesita permisiuni 'ROOT'. fisierul 'hosts' se afla in partitia /system care din start vine cu drepturi doar de citire.
